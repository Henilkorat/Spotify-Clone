# Task 6: Host a Static Website with GitHub Pages

## 📌 Objective
Deploy a simple website using **GitHub Pages**.

## 🛠️ Tools Used
- GitHub Pages

## 📂 Deliverables
- 🔗 **Live Website:** `https://henilkorat.github.io/Spotify-Clone/`

## 📝 Steps Followed

1. Created the website files (`index.html`, `script.js`, `css/`, `Assets/`, `songs/`).
2. Pushed the project to a new GitHub repository.
3. Went to repo → **Settings → Pages**.
4. Under **Build and deployment**, selected:
   - Branch: `main`
   - Folder: `/ (root)`
5. Saved settings — GitHub generated a live link.
6. Accessed the live website using the link provided by GitHub.
7. Customized the site further using CSS.

## ✅ Outcome
Successfully deployed a static website using GitHub Pages, accessible via a public live link.

---

## 🎵 About This Project

For this task, the website deployed is a basic **Spotify UI clone**, built with HTML, CSS, and JavaScript, recreating a simple music player interface (play, pause, track switching).

### Built With
- HTML5
- CSS3
- JavaScript

### Project Structure
```
├── Assets/          # Images, icons, and other static assets
├── css/             # Stylesheets
├── songs/           # Audio files used in the player
├── favicon.ico      # Website favicon
├── index.html       # Main webpage
├── script.js        # JavaScript functionality (player logic)
└── README.md        # Project documentation
```

### Features
- Spotify-like UI layout
- Play / Pause music functionality
- Song switching (next/previous)
